## use-dayjs

It's simpler to use dayjs

- Khai báo một lần, sử dụng mãi mãi
- Không bị lặp lại import moment ở mọi nơi
- Không bị lặp lại code ở mọi nơi
- Cơ động và thực chiến

## Support for

```bash
- Typescript
- React js
- Next js
- Node js
```

## Install package

```html
npm i --save use-dayjs
yarn add use-dayjs
```
